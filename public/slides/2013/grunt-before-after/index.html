Installation

(OS X/Homebrew)

$ brew install node
$ curl http://npmjs.org/install.sh | sh
$ npm install -g grunt-cli
$ npm install grunt --save-dev
$ npm install {what-you-want}
Ca ne doit pas être bien difficile de trouver pour Linux, peut être un peu plus pour Windows
